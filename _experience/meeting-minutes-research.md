## Overview

This research project focused on developing a Multi-Criteria Decision Analysis (MCDA) pipeline for automatic meeting minutes generation using state-of-the-art NLP models.

## Key Contributions

- Developed MCDA pipeline analyzing 2M+ data points for meeting minutes generation
- Achieved 128% better performance than BERT and T5 models using TOPSIS ranking
